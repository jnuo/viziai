🇹🇷 TÜRKÇE

ViziAI, kan tahlili sonuçlarınızı yapay zeka ile analiz eden ve sağlık trendlerinizi görselleştiren bir uygulamadır. Bu eklenti, e-Nabız sistemindeki tahlil sonuçlarınızı doğrudan ViziAI hesabınıza aktarmanızı sağlar.

KİMLER İÇİN?

• Türkiye'de yaşayıp tahlil sonuçlarını düzenli takip etmek isteyenler
• Yurt dışında yaşayıp ailesinin sağlık verilerini uzaktan takip edenler
• Birden fazla aile bireyinin tahlillerini tek panoda görmek isteyenler

NASIL ÇALIŞIR?

1. viziai.app adresinden hesabınıza giriş yapın
2. Ayarlar → API Anahtarları sayfasından bir anahtar oluşturun
3. Eklenti penceresine anahtarınızı yapıştırın ve bağlantıyı test edin
4. enabiz.gov.tr → Tahlillerim sayfasına gidin
5. Her tahlil sonucunun yanında "ViziAI'a Gönder" butonu görünecektir
6. Tek tıkla sonuçlarınız ViziAI'a aktarılır — onay sayfasında inceleyip kaydedin

İlk kez yüklediğinizde, e-Nabız sayfasının üstünde kurulum rehberi belirir. Adımları takip ederek 2 dakikada hazır olursunuz.

ÖZELLİKLER

• Tek tıkla aktarım — PDF indirmeye gerek yok
• Otomatik metrik tanıma — test adları, değerler, birimler ve referans aralıkları otomatik okunur
• Çift kayıt koruması — aynı tahlil tekrar aktarılamaz
• Güvenli bağlantı — verileriniz API anahtarı ile şifrelenerek iletilir
• Aktarım öncesi onay — sonuçları kaydetmeden önce inceleme imkânı

GÜVENLİK

• API anahtarınız yalnızca cihazınızda saklanır
• Sunucuda yalnızca anahtarın şifrelenmiş özeti tutulur
• Anahtarı istediğiniz zaman iptal edebilirsiniz
• Eklenti yalnızca enabiz.gov.tr ve viziai.app ile iletişim kurar

ViziAI hakkında daha fazla bilgi için: https://www.viziai.app

🇬🇧 ENGLISH

ViziAI is an app that analyzes your blood test results with AI and visualizes your health trends. This extension lets you import your lab results from e-Nabız (Turkey's national health portal) directly into your ViziAI account.

WHO IS THIS FOR?

• People living in Turkey who want to track their lab results regularly
• Diaspora families managing parents' health data remotely
• Anyone tracking multiple family members' results in one dashboard

HOW IT WORKS

1. Sign in at viziai.app
2. Go to Settings → API Keys and create a key
3. Paste the key into the extension window and test the connection
4. Navigate to enabiz.gov.tr → Tahlillerim (My Lab Results)
5. A "Send to ViziAI" button appears next to each lab report
6. One click imports your results — review and confirm on the approval page

When first installed, a setup guide appears at the top of the e-Nabız page. Follow the steps — you'll be ready in 2 minutes.

FEATURES

• One-click import — no need to download PDFs
• Automatic metric recognition — test names, values, units, and reference ranges are read automatically
• Duplicate protection — the same report cannot be imported twice
• Secure connection — your data is transmitted using an encrypted API key
• Pre-import review — review results before saving

SECURITY

• Your API key is stored locally on your device only
• The server stores only an encrypted hash of the key
• You can revoke your key at any time
• The extension only communicates with enabiz.gov.tr and viziai.app

Learn more: https://www.viziai.app

3. Fügen Sie den Schlüssel in das Erweiterungsfenster ein und testen Sie die Verbindung
